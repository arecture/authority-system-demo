# authority-system-demo

本项目为个人练手项目。目的是个人对Java后端开发的流程与编码更加熟悉

本项目为权限管理系统，旨在组织中进行后台系统权限管理。

本项目参考b站的项目：https://www.bilibili.com/video/BV1dt4y1x7rT

前端采用vue-admin模板进行修改

由于不会将前端项目和后端项目合并，所以删除了之前的仓库，重新提交

后端技术栈：

- Spring Boot
- MyBatis Plus
- Spring Security
- Jwt
- Redis
- MySQL

前端技术栈：

- HTML 5
- CSS 3
- JavaScript
- Vue
- Element UI
